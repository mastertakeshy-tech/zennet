Módulo 3 — Seguridad en Internet

La seguridad digital consiste en proteger nuestra información personal y navegar de forma segura en internet.


Riesgos en Línea

  • Phishing o correos falsos.
  • Robo de identidad.
  • Virus y malware.
  • Ciberbullying.

¿Qué es el Phishing?

El phishing es una técnica utilizada por ciberdelincuentes para engañar a las personas y robar información personal como contraseñas, cuentas bancarias o datos privados.

Normalmente se presenta mediante correos falsos, mensajes sospechosos o páginas web que parecen reales.


¿Qué es el Malware?

El malware es un software malicioso diseñado para dañar dispositivos, robar información o controlar sistemas sin permiso.

Los virus, spyware y troyanos son tipos comunes de malware.


¿Qué es el Robo de Identidad?

El robo de identidad ocurre cuando otra persona utiliza tus datos personales para hacerse pasar por ti.

Esto puede incluir cuentas falsas, robo de dinero o acceso ilegal a información privada.


Señales de Peligro

  • Mensajes con amenazas o urgencia.
  • Enlaces sospechosos.
  • Solicitudes de contraseñas.
  • Errores ortográficos en correos o páginas.

Consejos de Seguridad

✓ No compartir contraseñas.

✓ Revisar enlaces antes de hacer clic.

✓ Usar contraseñas seguras.

✓ No aceptar solicitudes de desconocidos.

✓ Activar autenticación en dos pasos.


Dato Importante

La mayoría de ataques digitales comienzan con correos falsos o mensajes engañosos.
